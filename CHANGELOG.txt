小兔猪 Live2D 控制台 - 版本更新记录

v1.0.0 (2026-07-14)
- 初始版本：Live2D 小兔猪模型展示 + 控制面板
- 支持6种表情、6种动作、5种耳朵、4种部件切换、2种变形、2种飞头
- 键盘快捷键控制
- 鼠标追踪/拖拽/缩放

v1.1.0 (2026-07-14)
- 新增聊天对话功能：底部聊天面板，可与小兔猪对话
- 接入火山方舟 API (doubao-seed-2-0-mini-260428)，流式输出
- AI 回复自动触发对应表情/动作/耳朵/切换，标签从显示文本中剔除
- API Key 管理：localStorage 存储，弹窗输入/修改
- 聊天面板可折叠，不影响模型展示
- 输入框聚焦时键盘快捷键自动屏蔽，避免误触
- AI 触发的动作同步更新右侧按钮 active 状态

v1.1.1 (2026-07-14)
- 修复 Retina 屏幕下模型模糊：PIXI resolution 设为 devicePixelRatio
- 修复聊天面板折叠/展开时模型被拉伸：聊天面板改为 absolute 浮层
- 修复输入框触发 Chrome 密码管理器弹窗：添加 autocomplete=off + readonly 技巧

v1.3.0 (2026-07-14)
- API Key 迁移到服务端：key 存在 Node proxy 环境变量，浏览器不再发送
- 移除前端 API Key 输入弹窗和 localStorage 存储
- 修复流式返回：解析 reasoning_content 字段，添加 reasoning_effort: minimal 加速
- 添加 max_tokens: 1024, temperature: 0.7 优化响应速度
- 修复进度条卡在10%：增加假进度动画，加载过程更流畅
- 修复线上无流式返回：Nginx -> Node SSE proxy -> 火山方舟，避免 Nginx HTTPS 代理缓冲 SSE
- 小兔猪说话时嘴巴会动（lip sync）：AI 流式回复期间嘴巴随机开合，回复结束闭合
- Loading 页面增加进度条和百分比显示
- 聊天输入框改为 contenteditable div，彻底解决 Chrome 密码选择器弹窗
